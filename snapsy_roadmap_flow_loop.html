Improved wizard: before vs after

Current — 6 steps

1. Event name
2. End date
3. Reveal timing
4. Camera style
5. Guest count + shots
6. Invite card design
Plain text modal: code + URL

Improved — 4 steps

0. Event type picker (new)
1. Event name (pre-suggested from type)
2. Date + reveal timing (merged)
3. Camera style
4. Guest count + shots + value math
Invite card design (inline, skippable)
WhatsApp share sheet — visual card
